# 🚀 Modern Monthly Habit Tracker

A sleek, user-friendly habit tracker built with **Vanilla JavaScript**, featuring a modern **Glassmorphism UI** and persistent data storage. This application is fully responsive and optimized for both Desktop and Mobile devices.

## ✨ Key Features
* **Glassmorphism Design:** Premium UI with blurred backgrounds and soft gradients.
* **Smart Grid Layout:** Automatically adjusts the number of cards based on your screen size (Laptop vs. Mobile).
* **Dark Mode Toggle:** Seamless theme switching with a custom-designed teal toggle.
* **Persistent Storage:** Uses `localStorage` to keep your habits saved even after refreshing the page.
* **Interactive 31-Day Grid:** Dynamically generates days based on the selected month.
* **Habit Management:** Easily add and delete habits for every specific day.

## 📱 Responsive Preview
* **Laptop:** Multi-column grid for wide viewing.
* **Mobile:** Single-column stack for easy thumb-scrolling and interaction.

## 🛠️ Tech Stack
* **HTML5** - Semantic structure.
* **CSS3** - Custom properties (variables), Flexbox, CSS Grid, and Media Queries.
* **JavaScript (ES6)** - DOM Manipulation, LocalStorage API, and Theme logic.
